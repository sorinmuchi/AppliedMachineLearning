README

File: knn.py
Purpose: classify data using the k-Nearest-Neighbor algorithm, and
		 print confusion matrix
Requirements:
	- ./data/
	- ./data/train_input.csv
	- ./data/test_input.csv
	- ./data/train_output.csv

Expected output:
	- iteration number n, where n = [0,n)
	- confusion matrix

How to run:
	pthon knn.py
